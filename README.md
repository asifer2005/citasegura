# CitaSegura

**Menos ausencias y agenda llena: confirmaciones, lista de espera y señal anti no-show.**

🔗 **Demo en vivo:** https://asifer2005.github.io/citasegura/

## Qué resuelve
Cada hueco que deja un cliente que no aparece es dinero perdido. CitaSegura confirma las citas por WhatsApp, rellena los huecos con lista de espera automática y permite cobrar una señal para que los clientes no falten.

## Funciones
- Confirmaciones por WhatsApp/SMS ya escritas.
- Lista de espera automática que rellena huecos.
- Señal anti no-show para asegurar la cita.
- Panel de no-shows y dinero recuperado.
- Envío por lotes y varios profesionales en Pro.
- Muro de suscripción (Free / Pro / Business) con checkout simulado.

## Tecnología
Aplicación web de un solo archivo (`index.html`), HTML + CSS + JavaScript _vanilla_, sin dependencias ni build. Los datos persisten en `localStorage`, así que funciona offline abriendo el archivo en el navegador.
